Don't you fret!

Color theory is complicated, but you don't have to be an expert in hues saturations and tones it to apply it. There are many free palettes to use, or tools to create your own!

The simplest method for adding color is to type the color name you are looking for directly into the code as shown under the Color Name heading. However, this method is quite limiting, with only 140 colors to choose from.

Your other option are choosing a HEX code, or RBG. These methods use a combination of the primary colors, Red Blue and Green, to change the hues. RGB assigns a number to each of the primary colors, whereas HEX condenses them into one 6 character code (8 digits if including transparency) proceeded by a #.
